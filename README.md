# Clouds-PHP-technical

## Requirements
- PHP ^8.1
- Laravel 10.0+
- Mysql

## Installation
- 1. download project
- 2. migrate
```bash
php artisan migrate
```
- 3. make queue:work
```bash
php artisan queue:work
```
- 4. Import Postman

## Postman
- 1. Register
- 2. Log In
- 3. Add Post
- 4. Show Post
- 5. Get All Post
